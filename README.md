# Do-it-JS-Study

## 스터디 주차별 예제 코드입니다.
받고싶은 폴더에 clone하셔서 사용하면 됩니다. 

## 실행방법
```git clone https://github.com/CodeBlue007/Do_it_JS_Study.git ```

## Update된 자료 받을때
```git pull origin main ```


코드 실행을 위해서는 

vsCode Code Runner라는 extension을 설치해야 합니다 !!

전부다 실행하면 오류뜨니까 > 실행하고 싶은 부분만 드래그 후 

```ctrl+alt+ N ``` 으로 실행가능합니다!!
